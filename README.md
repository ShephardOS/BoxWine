## About

**BoxWine** is a project designed to run Windows x86 applications in **Termux** using **Box86**, **Box64**, and **Wine**.

## System Requirements

- **OS**: Android 10+
- **Processor**: Snapdragon 665 or higher
- **RAM**: 4GB+ recommended
- **Storage**: At least 2GB of free space
- **Termux & X11**: Installed

## Installation
